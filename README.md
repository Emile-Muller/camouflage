# Camouflage

A **mobile-first role deduction game** designed to be played **on a single phone with a group of friends**.

Players take turns passing the phone around to secretly view their role, then put the phone down and dive into discussion, bluffing, and accusations. After each round, the group votes to eliminate a player, until the game reaches its win condition.

Think classic social deduction... but **no accounts, no downloads, no setup hassle**.

👉 **Free to play**  
👉 **Open source**  
👉 **Built for mobile, designed for groups**

## 🕹️ How It Works

1. One phone is used for the entire group
2. Players pass the phone around
3. Each player secretly views their role
4. The group discusses and debates
5. Players vote to eliminate someone
6. Repeat until the game ends

The interface is intentionally simple and touch-friendly, so it works smoothly in real-life group settings.

## 🛠️ Tech Stack

This project uses a modern, lightweight frontend stack focused on performance, maintainability, and developer experience.

### Frontend

- **React** — component-based UI
- **TypeScript** — static typing for safer, more maintainable code
- **Vite** — fast development server and optimized builds
- **Tailwind CSS** — utility-first styling for rapid UI iteration

### Internationalization

- **i18n** — handles translations and multilingual support (English & French)

- **Husky** — Git hooks for automated checks

This setup keeps the codebase clean, fast, and easy to contribute to.
